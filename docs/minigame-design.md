# 직장인 잔혹사 — TOP 10 미니게임 상세 기획서

## 공통 규칙
- **함정(Troll)**: 화면에 보이는 "당연한" 버튼/선택지는 전부 함정
- **클리어 액션**: 주변 환경을 관찰하고 기발한 방법을 찾아야 클리어
- **핵심 철학**: "이 게임은 보이는 대로 믿으면 안 된다"

---

## 1. [출근길] 알람 0.1초 컷 — 스마트폰 격파
**씬 키**: `AlarmScene`

**화면 뷰**: 1인칭 시점. 어두운 방 안, 눈앞에 스크린이 켜진 스마트폰. 그 뒤로 협탁(침대 옆 탁자)이 희미하게 보임.

**디자인 요소**:
- 스마트폰 UI: 화면 중앙에 거대한 [중지] 버튼과 [스누즈(5분 뒤)] 버튼
- 주변 환경(협탁): 물이 담긴 유리컵, 안경, 그리고 뜬금없이 놓여있는 '망치'(주변 그림자에 살짝 묻힘)

**플레이 액션**:
- **함정**: [중지] 버튼 → 이리저리 도망 다님. [스누즈] → 알람 소리 2배. 유리컵 클릭 → 물 엎질러져 감전(실패)
- **클리어**: 배경의 '망치'를 길게 꾹 눌러(Drag) 스마트폰 액정 정중앙에 가져다 놓음(Drop)
- **연출**: 콰직! 액정 박살 → 화면 암전 → 완벽한 정적

---

## 2. [생존기] 메신저 오타 — 물리적 서버 다운
**씬 키**: `BootingScene`

**화면 뷰**: 사무실 자리. 모니터에 사내 메신저 창이 떠 있음.

**디자인 요소**:
- 화면 중앙: 상사에게 보낸 "부장님 개짜증나" 말풍선 옆에 숫자 '1'. 말풍선 꾹 누르면 [삭제] 버튼 팝업
- 화면 하단: 모니터 밑으로 책상 밑 공간(콘센트, 랜선, 텀블러, 쓰레기통)이 스와이프하면 보임

**플레이 액션**:
- **함정**: [삭제] 버튼 연타 → "서버 지연 중..." 모래시계만 뜸. 키보드 치면 "뭐가 짜증 난다고?" 답장 옴(실패)
- **클리어**: 화면을 위로 쓸어 올려(Swipe Up) 책상 밑 시야 이동 → 모니터 연결된 '굵은 검은색 전원 플러그' 터치해서 뽑기
- **연출**: 찌직- 소리와 함께 모니터 화면이 시커멓게 꺼지며 안도의 한숨

---

## 3. [점심] 가위바위보 — 을의 생존법
**씬 키**: `MenuRouletteScene`

**화면 뷰**: 1인칭 시점. 카페 카운터 앞. 부장님이 등을 보인 채 서 있음.

**디자인 요소**:
- 부장님: 등을 보인 채 가위바위보 손을 뒤로 빼서 준비 동작
- 배경: 부장님 정면의 반짝이는 '에스프레소 머신 거울(반사면)'
- 하단 UI: ✌️(가위), ✊(바위), ✋(보) 버튼 3개

**플레이 액션**:
- **함정**: 거울 안 보고 찍어서 이기면 "네가 감히 이겨?" 분노(실패). 비기면 무한 반복
- **클리어**: 에스프레소 머신 거울에 비친 부장님 손 모양 확인 → 일부러 지는 패 클릭
- **연출**: "껄껄, 내가 냈지! 커피 사 와!" 법인카드 건네줌

---

## 4. [회의실] 빈말 퍼레이드 — 영혼 없는 앵무새
**씬 키**: `SleepFightScene`

**화면 뷰**: 회의실 1인칭 시점. 정면에 부장님이 열변을 토하는 중.

**디자인 요소**:
- 부장님 말풍선: "요즘 MZ들은 말이야..." (점점 글씨 많아짐)
- 하단 UI: 4개의 대답 선택지 버튼 (메신저 자동완성 버튼 스타일)

**플레이 액션**:
- **함정**: 4개 중 1개는 '팩트 폭행("그건 아닙니다")', 1개는 '딴소리("점심 뭐 먹지")'. 누르면 레이저 눈빛(실패)
- **클리어**: 가장 AI 같고 영혼 없는 대답만 빠르게 3회 연속 클릭 ("좋은 인사이트입니다!", "제가 팔로업하겠습니다", "역시 부장님이십니다")
- **연출**: 3콤보 성공 시 "역시 김대리는 내 마음을 잘 알아" 흐뭇하게 끄덕임

## 5. [보고서] 저장의 저주 — 불변의 법칙
**씬 키**: `FileSaveScene`

**화면 뷰**: PC 바탕화면 폴더 창.

**디자인 요소**:
- 중앙: 커다란 워드(Word) 아이콘. 파일명 `보고서_최종_진짜최종_이게마지막.docx`
- UI: 파일명 클릭 시 모바일/가상 키보드가 하단에서 올라옴

**플레이 액션**:
- **함정**: 이름을 지우고 다른 .docx로 수정해서 엔터 → 파일이 2개, 4개로 무한 증식
- **클리어**: 백스페이스로 `.docx` 부분만 지운 뒤, `.pdf` 타이핑하고 엔터
- **연출**: 아이콘이 워드(파란색)→PDF(빨간색 자물쇠)로 철컥! 변환. "음, 더 수정할 수가 없군. 결재!"

## 6. [딴짓] 웹서핑 숨기기 — 물리적 차단
**씬 키**: `AltTabScene`

**화면 뷰**: 1인칭 시점. 모니터에 화려한 쇼핑몰 창.

**디자인 요소**:
- 화면 중앙: 모니터 속 쇼핑몰 UI
- 화면 외곽: 베젤(모니터 테두리), 우측 하단에 물리 전원 버튼(LED 빛남)
- 긴장감: 좌측에서 부장님 시커먼 실루엣(그림자)이 점점 다가옴
- 가짜 UI: 화면 정중앙에 깜빡이는 [Alt+Tab] 아이콘

**플레이 액션**:
- **함정**: [Alt+Tab] 누르면 윈도우 모래시계 뜨며 렉. 부장님 그림자 도착(실패)
- **클리어**: 부장님 그림자가 모니터 덮기 직전, 베젤의 '물리 전원 버튼' 클릭
- **연출**: 팟! 물리적으로 까맣게 꺼짐. "어? 김대리 컴퓨터 꺼졌네? 쉬어~"

---

## 7. [빌런 대처] 다리 떠는 빌런 — 친환경 에너지
**씬 키**: `ChargingScene`

**화면 뷰**: 옆자리 동료 하반신과 내 책상이 절반씩 보임.

**디자인 요소**:
- 동료: 미친 듯이 다리를 떨음 (화면 전체 미세하게 흔들림)
- 내 책상: 배터리 1% 스마트폰, 살짝 열린 서랍 안의 '소형 발전기', 'USB 케이블', '스테이플러'

**플레이 액션**:
- **함정**: 빌런 다리를 꾹 누르면 멈추지만 떼면 다시 떪. 스테이플러를 다리에 드래그 → 인사팀 호출(실패)
- **클리어**: 서랍의 '소형 발전기' → 빌런 무릎에 부착 → 'USB 케이블'로 발전기와 스마트폰 연결
- **연출**: 덜덜거리는 동력으로 발전기 돌아감 → 배터리 1%→100% 초고속 충전(띠링~)

## 8. [야근] 가족 거짓말 — AI 오토 파일럿
**씬 키**: `MosquitoScene`

**화면 뷰**: 카카오톡 대화방 화면. 폭탄 카톡 쏟아짐.

**디자인 요소**:
- 하단 입력창: 유저가 글씨를 칠 수 있는 가짜 입력창
- 상단 UI: 카톡방 우측 상단의 [≡ 메뉴] 버튼과 화면 구석의 톱니바퀴 [설정] 아이콘

**플레이 액션**:
- **함정**: 입력창에 타이핑 → 자동완성으로 "오늘 집에 못가, 퇴사할래" 등 극단적 오타(전송 시 실패)
- **클리어**: 상단 [설정] 톱니바퀴 → 팝업에서 [야근용 챗봇 AI: OFF] → [ON]으로 변경
- **연출**: "사랑하는 여보♥ 금방 마무리하고 갈게!" + 이모티콘 자동 발송, 가족 "화이팅!" 답장

## 9. [회식] 술자리 탈출 — 영수증 밧줄
**씬 키**: `SomekScene`

**화면 뷰**: 회식 장소 식당 화장실. 방탈출 게임 시점.

**디자인 요소**:
- 배경: 세면대, 높은 곳의 작은 환기 창문
- 아이템: 화장지, 뚫어뻥, 바지 주머니의 '영수증 뭉치', 세면대 위 '청테이프'

**플레이 액션**:
- **함정**: 화장지 → 창문에 걸면 찢어짐. 뚫어뻥 → 튕겨 나옴. 문으로 나가기 → 부장님 등장(실패)
- **클리어**: '영수증 뭉치' 획득 → '청테이프' 획득 → 인벤토리에서 두 개 합성(드래그 앤 드롭) → '영수증 밧줄' 완성 → 창문에 드래그
- **연출**: 영수증들이 끝없이 이어져 밧줄 됨 → 닌자처럼 창밖 하강 탈출

## 10. [퇴사] 사직서 투척 — 비행기 날리기
**씬 키**: `ResignScene`

**화면 뷰**: 부장님 책상 앞. 1인칭 시점 앵그리버드 뷰.

**디자인 요소**:
- 정면: 부장님이 입을 벌리고 꿀잠 자는 중
- 하단: 내 손에 들린 하얀색 '사직서 봉투'

**플레이 액션**:
- **함정**: 사직서 봉투를 그대로 드래그해서 던지면 얼굴에 맞고 튕김 → 부장님 깨서 사직서 찢음(실패)
- **클리어**: 사직서 봉투를 좌우로 슬라이드(문지르기) → 종이비행기로 변신 → 뒤로 당겼다 놓아(Slingshot) 부장님 입 정중앙으로 발사
- **연출**: 입속으로 쏙 들어감 → 무의식중에 꿀꺽 → 도장 쾅! "퇴사 승인!" 타이포그래피
