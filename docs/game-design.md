# 직장인 잔혹사 — 게임 디자인 문서

## 개요

- **장르**: 미니게임 모음 (WarioWare 스타일)
- **플랫폼**: 웹 (모바일 WebView / Steam Electron)
- **엔진**: Phaser 3 + React + TypeScript
- **컨셉**: 직장인의 하루를 10개 스테이지로 표현. 각 스테이지마다 10개의 랜덤 미니게임 중 1개가 출제됨.

## 게임 흐름

```
타이틀 → 스테이지 선택 (순차 해금) → 미니게임 인트로 → 미니게임 플레이 → 결과 → 스테이지 선택
                                                                                    ↓ (10개 클리어)
                                                                                엔딩 화면
```
